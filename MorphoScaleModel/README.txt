フォルダ名について
w : ２つのレイヤの横幅[nm]
d : ２つのレイヤの厚さ[nm]
symm : 左右対称かどうか
例)
w300_500_d100_150
=> 横幅は媒質1が300nm 媒質2が500nm, 厚さは媒質1が100nm 媒質2が150nm, 左右対称でない という意味

w300_500_d100_150_symm
=> 横幅は媒質1が300nm 媒質2が500nm, 厚さは媒質1が100nm 媒質2が150nm, 左右対称である(単純な多層膜) という意味

n : 2つのレイヤの屈折率
m : レイヤの枚数
r : 頂上に置ける横幅の縮小率

例)
n1.56_1.00_m8_r0.30
=> 屈折率は媒質1が1.56, 媒質2が1.0(空気), レイヤの枚数は8, 頂上に行くにつれ横幅は狭くなり最終的には30%に縮小されるモデルとなる.
